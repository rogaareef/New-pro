❓ Frequently Asked Questions

How do I get an Anthropic API key?

Visit console.anthropic.com, create a free account, and generate an API key. The key starts with sk-ant-. Paste it in any tool on ProScholarTools to get started.

Is ProScholarTools completely free?

Yes. All 22 tools are free. You only need your own Anthropic API key — which has a free tier. We are funded by Google AdSense advertising.

Is my API key secure?

Yes. Your API key is stored only in your browser's localStorage. It never reaches our servers. You can clear it at any time from the tool settings.

Do you support Arabic?

Yes. ProScholarTools is fully bilingual with complete Arabic RTL support. Use the language toggle in the top right of any page.

I found a bug — how do I report it?

Email us at gogo24173@gmail.com with details of the issue including your browser, device, and steps to reproduce.

Can I suggest a new tool?

Absolutely. Email your suggestion to gogo24173@gmail.com — we review all suggestions and prioritize based on community demand.
